**Автоматизация расчётов в квантовой механике** — оконное приложение для выполнения ключевых вычислений квантовой физики.  
Программа позволяет быстро рассчитывать энергетические уровни, волновые функции и вероятности переходов, наглядно отображая результаты и исключая арифметические ошибки при решении типовых учебных задач.

## Использование

### Обзор возможностей

Приложение решает шесть типовых задач по квантовой механике:

- **Энергия частицы в бесконечно глубокой прямоугольной потенциальной яме** — расчёт уровней энергии `En` для стационарных состояний.
- **Волновая функция гармонического осциллятора** — вычисление `ψ(x)` и плотности вероятности для основного и возбуждённых состояний.
- **Коэффициент прохождения через прямоугольный потенциальный барьер** — туннельный эффект, расчёт вероятности прохождения.
- **Энергия основного состояния в треугольной потенциальной яме** — численное решение уравнения Шрёдингера.
- **Вероятность квантового перехода** под действием зависящего от времени возмущения.
- **Средние значения операторов координаты и импульса** для заданной волновой функции.

Все вычисления сопровождаются проверкой вводимых данных (числовой формат, физически допустимые диапазоны).

### Системные требования

- **ОС** — Windows 7 / 8 / 10 / 11 (32- или 64-бит). При использовании .NET 6+ возможна работа на Linux и macOS.
- **Процессор** — от 1 ГГц
- **ОЗУ** — 2 ГБ
- **Свободное место на диске** — 500 МБ
- **Дополнительно** — .NET Framework 4.7.2 или новее (обычно предустановлен)

### Установка

1. Скачайте последнюю версию программы из [репозитория GitHub]([https://github.com/yourusername/QuantumPhysics]).
2. Убедитесь, что в системе присутствует .NET Framework 4.7.2+. Если нет — загрузите с [официального сайта Microsoft](https://dotnet.microsoft.com/).
3. Распакуйте архив и запустите `QuantumPhysics.exe` (ярлык можно вынести на рабочий стол).

### Запуск

Откройте исполняемый файл или созданный ярлык. Появится главное окно с перечнем доступных задач.

### Работа с программой

1. **Выбор задачи** — установите переключатель напротив нужного расчёта (1–6) и нажмите кнопку «Перейти к задаче».
2. **Ввод данных** — заполните поля необходимыми параметрами (масса частицы, ширина ямы, номер уровня и т.п.). При некорректном вводе появится предупредительное сообщение.
3. **Расчёт** — нажмите кнопку «Рассчитать». Результат отобразится в текстовом поле, а при необходимости — в виде графика.
4. **Сброс** — кнопка «Сброс» очищает все поля ввода.
