**Вычисления в электродинамике** — оконное приложение для автоматизации расчётов по шести базовым формулам постоянного электрического тока.  
Программа мгновенно выдаёт точный результат, исключая арифметические ошибки и экономя время при выполнении типовых учебных задач.

## Использование

### Обзор возможностей

Приложение решает шесть типовых задач по электродинамике:

- **Закон Ома для участка цепи** — расчёт силы тока, напряжения или сопротивления:  
  `I = U / R`
- **Мощность электрического тока**:  
  `P = U · I`
- **Работа электрического тока**:  
  `A = U · I · t`
- **Сопротивление проводника** через удельные параметры:  
  `R = ρ · l / S`
- **Общее сопротивление при последовательном соединении**:  
  `R = R₁ + R₂ + … + Rₙ`
- **Общее сопротивление при параллельном соединении** (для двух резисторов):  
  `R = (R₁ · R₂) / (R₁ + R₂)`

Все вычисления сопровождаются проверкой корректности ввода (числовой формат, допустимые значения).

### Системные требования

- **ОС** — Windows 7 / 8 / 10 / 11 (32- или 64-бит)
- **Процессор** — от 1 ГГц
- **ОЗУ** — 1 ГБ (32-бит) / 2 ГБ (64-бит)
- **Свободное место на диске** — 16–20 ГБ (требования ОС)
- **Дополнительно** — .NET Framework 4.7.2 или новее (обычно предустановлен)

### Установка

1. Скачайте последнюю версию программы из [репозитория GitHub]([https://github.com/dima09328678-beep/kursovaya-rabota]).
2. Убедитесь, что в системе присутствует .NET Framework 4.7.2+. Если нет — загрузите с [официального сайта Microsoft](https://dotnet.microsoft.com/).
3. Распакуйте архив и запустите `ElectrodynamicsCalc.exe` (ярлык можно вынести на рабочий стол).

### Запуск

Откройте исполняемый файл или созданный ярлык. Появится главное окно с перечнем доступных задач.

### Работа с программой

1. **Выбор задачи** — нажмите кнопку, соответствующую нужному расчёту (Закон Ома, Мощность, Работа, Сопротивление проводника, Последовательное или Параллельное соединение).
2. **Ввод данных** — заполните поля необходимыми числовыми значениями (напряжение, сила тока, сопротивление, длина, площадь сечения и т.п.).
3. **Расчёт** — нажмите кнопку «Рассчитать». При некорректных данных (пустые поля, отрицательные числа, буквы) появится предупредительное сообщение.
4. **Результат** — итоговое значение отобразится в поле «Ответ».
